# namnlista

Lookup swedish name days.

## usage

```javascript
import nameday from 'namnlista'

nameday.lookup(new Date())
```

## api

### `.lookup([ name || date ])`

### `.all`

### `.external`

## cli usage

```
  Usage: namnlista [options] [command]

  Options:
    -V, --version  output the version number
    -h, --help     output usage information

  Commands:
    today [options]
    date [options] <month> <date>
    external [options] <provider>
    providers
```
